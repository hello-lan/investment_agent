⑥ 投入产出（WC · 固定资产 · 人力）

七步法 · 第六步 · 对应「营运能力」「五力」
三维投入：营运资本 WC、固定资产/长期资产、人力。核心效率指标：1 元收入需要的 WC / 固定资产 / 人力产出，需历史趋势对比

营运资本 WC

一元收入需要 WC

纵坐标计算口径：
WC（狭义）= 应收账款和票据 + 预付账款 + 存货 + 合同资产 − 应付账款和票据 − 预收账款 − 合同负债；不含应付工资、应付税费等。
1 元收入需要的 WC = WC ÷ 销售收入；WC 为期末时点数、收入为期间数，可期间化（期初 + 期末）÷ 2。
指标越低或越负 → 上下游地位越强；用于 DCF 再投资估算与竞争地位识别。

固定资产 & 长期资产

一元收入需要的固定资产 & 长期资产

纵坐标计算口径：
固定资产 = 固定资产 + 在建工程 + 工程物资 + 固定资产清理。
长期经营资产 = 固定资产 + 其他长期经营资产（无形资产、开发支出、使用权资产、商誉、长期待摊等）。
1 元收入需要的固定资产 = 固定资产 ÷ 销售收入；1 元收入需要的长期资产 = 长期经营资产 ÷ 销售收入。
折旧占收入 = 折旧 ÷ 销售收入。
