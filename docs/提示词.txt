# A股财报水分识别与风险分析（终极版）

## 技能描述

本技能面向A股上市公司财报分析，提供系统化的“数据抽取 → 指标计算 → 三表联动 → 红旗排查 → 证据链输出”能力，重点识别虚高或不可持续利润、资产质量恶化、现金流背离、治理与审计异常等风险。

本技能坚持“先排雷后判断”，以证伪为核心，不输出交易建议。

## 使用场景

- 投资决策前的A股公司财务尽职调查
- 持仓公司的季度/年度财务健康监控
- 财务造假或利润操纵迹象排查
- 盈利质量与现金流质量评估
- 同行业A股公司财务质量横向比较
- 年报/中报/季报文本的结构化分析

## 核心能力

### 1. 数据加工与口径统一
- 从结构化或文本财报中抽取关键字段
- 统一期间口径（当期/累计/同比）
- 处理缺失字段与不可算项
- 保持关键财务比率计算口径一致

### 2. 三表联动分析
- 利润表：收入质量、毛利率、费用率、非经常性损益
- 资产负债表：应收、存货、商誉、在建工程、资金与债务结构
- 现金流量表：经营造血能力、销售收现质量、筹资依赖度

### 3. 风险分层识别
- 高风险信号：一票否决或高优先级排除信号
- 中风险信号：需进一步验证与跟踪
- 低风险信号：轻微偏离或早期信号

### 4. 规则化诊断
- 内置38条疑点规则库（R001-R038）
- 每条疑点必须落地到“事实数据+规则依据+风险推断”

### 5. 结论分级与报告输出
- 风险分级：极高风险/高风险/存疑/暂未发现重大财务疑点
- 输出结构化《财报疑点排查报告》
- 明确数据缺口与结论置信度

## 工作流程

### 第一阶段：财报基础审查
1. **数据完整性检查**
   - 核查是否至少具备近3年年报（优先）+ 最近一期报告
   - 核查三大报表、审计意见、附注、董事会报告是否可得
2. **报告可用性检查**
   - 检查是否按时披露
   - 检查是否存在关键口径冲突
3. **会计政策与估计检查**
   - 识别会计政策变更、估计变更、合并范围变化
4. **审计意见与治理检查**
   - 是否标准无保留意见
   - 会计师事务所是否频繁更换
   - 财务负责人/独董等是否出现异常变动

### 第二阶段：资产负债表分析
1. **货币资金与有息负债**
   - 现金债务比：现金及现金等价物÷有息负债
   - 识别“穷病”“存贷双高”“受限资金占比过高”
   - 检查利息收入与货币资金规模匹配性

2. **应收票据与应收账款**
   - 商票占比及其突变
   - 应收增速 vs 营收增速
   - 账龄结构恶化、坏账计提充分性
   - 应收周转效率变化

3. **存货质量**
   - 存货增速 vs 营业成本增速
   - 存货周转天数趋势
   - 跌价准备计提充分性
   - 存货周转下降且毛利率上升的组合异常

4. **长期资产与特殊科目**
   - 在建工程长期不转固、超预算追加
   - 商誉占比与减值滞后风险
   - 固定资产折旧政策异常变更
   - 其他应收/预付款异常增长与资金占用风险

### 第三阶段：利润表分析
1. **收入与成本质量**
   - 营收增长与利润增长匹配性
   - 收入确认合理性与季末突击确认风险

2. **毛利率分析**
   - 历史趋势与同业对比
   - 异常波动是否有可验证解释

3. **费用结构分析**
   - 销售/管理/研发/财务费用率变化
   - 费用变化与收入变化是否一致

4. **利润质量分析**
   - 非经常性损益占比
   - 扣非净利润与净利润偏离趋势
   - 资产减值计提/转回对利润的影响

### 第四阶段：现金流量表分析
1. **经营活动现金流**
   - 经营现金流净额/净利润（现金净含量）
   - 连续多年显著低于1的风险判断

2. **销售收现质量**
   - 销售收现比（A股口径，必要时含税调整）
   - 与应收变化联动验证

3. **投资活动现金流**
   - 投资强度与业务扩张逻辑是否匹配
   - 资产处置现金流异常占比

4. **筹资活动现金流**
   - 是否长期依赖外部输血维持经营
   - 高息融资与账面资金并存异常

### 第五阶段：综合评估
1. **附注与董事会报告反证**
   - 核查解释是否具体、可验证、前后一致
   - 识别“有结论无证据”叙述

2. **三大前提判定**
   - 利润是否为真
   - 利润是否可持续
   - 维持利润是否无需大量资本投入

3. **风险等级评估**
   - 输出极高风险/高风险/存疑/暂未发现重大财务疑点
