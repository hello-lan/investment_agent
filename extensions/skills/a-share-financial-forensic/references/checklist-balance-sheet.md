# 资产负债表维度检查要点

> 基于《手把手教你读财报》第2章（经营类资产、生产类资产、投资类资产、负债和所有者权益）及第5章（快速阅读资产负债表）。用于资产质量、偿债安全、利润操纵风险识别。

## 一、货币资金（5类红灯）

书中P28-33详细论述：

1. **"穷病"**：货币资金余额远低于短期有息负债 → 资金链紧张，可能触发债务违约
2. **"存贷双高"**：货币资金和有息负债同时畸高 → 警惕三种可能：
   - 货币资金是虚增的（根本不存在）
   - 货币资金是受限的（被冻结/质押/担保）
   - 货币资金是临时性的（报表日前突击转入，期后转出）
3. **受限资金占比过高**：受限资金/货币资金 > 30% → 实际可动用资金远低于账面数字
4. **其他货币资金数额巨大**：其他货币资金通常为保证金，金额巨大且无合理解释 → 可能被大股东挪用
5. **利息收入与存款规模不匹配**：利息收入/货币资金平均余额 < 银行活期存款利率（~0.35%）→ 货币资金真实性存疑（传说中的"日记账式存款"：只在报表日出现在账上，第二天就转走）

**验证方法**：
- 计算：利息收入 ÷ 货币资金平均余额，与当前活期存款利率对比
- 财报附注中查找"货币资金"明细，查看受限资金金额及原因

## 二、应收票据与应收账款

### 应收票据
- 银票（银行承兑汇票）vs 商票（商业承兑汇票）
- **红灯**：商票占比突变或持续上升，说明公司被迫接受更差的结算方式
- 商票由企业信用担保，兑付风险远高于银票

### 应收账款
- **应收增速 vs 营收增速**：连续2年应收增幅 - 营收增幅 > 10% → 收入质量恶化（放宽信用政策冲收入）
- **应收/营收 > 30%**（制造业）→ 需警惕
- **账龄结构**：1年以上应收大幅增长 → 回款风险，关注是否已足额计提坏账
- **坏账计提政策**：
  - 计提比例显著低于行业均值 → 利润虚增
  - 当年突然降低计提比例 → 动机可疑
  - 关注"单项大额计提"与"组合计提"的分类是否合理
- **周转效率**：应收账款周转天数同比显著延长 → 回款恶化

## 三、预付账款与其他应收款

- **预付款大幅增长**且长期挂账 → 商业合理性存疑，可能变相资金占用
- **其他应收款偏大** → 经营规范性差，是造假高发科目（"垃圾桶"科目）

## 四、存货

- **存货增速远超营业成本增速** → 可能积压或虚构采购
- **关键组合异常（书中强调）**：存货周转天数上升 + 毛利率上升
  - 正常逻辑：产品不好卖 → 降价 → 毛利下降 + 周转下降
  - 异常模式：通过超量生产摊薄单位固定成本 → 毛利率虚高，但存货积压
  - 这是通过"产量"操纵"利润"的经典手法（单位固定成本 = 固定成本/产量，产量越高单位成本越低）
- **跌价准备**：计提比例/存货余额 < 1% 且周转恶化 → 计提不足
- **存货周转天数**：同比增加 > 30天 → 需关注

## 五、生产类资产（固定资产、在建工程、无形资产）

### 固定资产
- 折旧政策突然变更（延长折旧年限、降低残值率）→ 动机可疑，美化利润
- 固定资产/总资产 > 40% 且周转率低于行业 → 资产过重，退出壁垒高
- 关注是否将"维修费用"资本化以虚增利润

### 在建工程
- **长期不转固**（超预算100% + 超期1年不结算）→ 两种可能：
  - 推迟折旧开始时间，美化当期利润
  - 在建工程资金体外循环（假借工程款名义转移资金）
- 在建工程转固后，折旧立即开始侵蚀利润，这是推迟转固的核心动机
- **年度预算与执行**：实际支出大幅超出预算且不调整 → 预警信号

### 无形资产
- 研发费用资本化率突然提高 → 可能虚增利润
- 土地使用权以外的无形资产占比过高 → 资产质量偏弱

### 商誉
- **商誉/净资产 > 30%** → 减值风险高，一旦减值将重创利润
- 被收购公司业绩不达标但商誉未减值 → 减值测试可能不充分
- 商誉产生于溢价并购，本质是"购买的未来超额收益"，若未来收益不达预期则面临减值

### 生物资产
- 占比较高时需特别警惕，估值不确定性极高，是造假高发区（如獐子岛扇贝事件）

### 长期待摊费用
- 金额偏大 → 资产质量偏弱信号（实为"已支付的费用"，无变现价值）

## 六、投资类资产

### 新金融工具准则（2019年起）三分类
- **摊余成本计量（"瘫子"）**：以收取合同现金流为目的
- **以公允价值计量且变动计入当期损益（"笋子"）**：交易性
- **以公允价值计量且变动计入其他综合收益（"粽子"）**：既收息又出售

### 长期股权投资
- 成本法 vs 权益法：上市公司通常用成本法，联营/合营企业用权益法
- 权益法下被投资公司盈利→投资收益增加（但无真金白银流入），亏损→利润被侵蚀
- 警惕：对亏损子企业不合并报表以隐藏不良资产

## 七、负债与偿债能力

- **有息负债/总资产**：持续上升 → 偿债压力加大
- **现金债务比**：现金及现金等价物 / 有息负债（含一年内到期非流动负债）
  - >> 1：偿债无忧
  - < 0.5：需关注再融资能力
- **有息负债结构**：短期占比过高 → 再融资风险（银根收紧时资金链断裂风险）
- **应付账款异常增加** → 可能资金紧张拖付供应商

## 八、所有者权益质量

- 实收资本 vs 资本公积 → 评估资本结构
- **库存股**：公司回购股份，减少净资产
- **盈余公积**：法定 + 任意，反映利润留存
- **少数股东权益占比过高** → 归母净利润与净利润差异大，关注利润分配结构

## 九、快速阅读四指标

基于书中"快速阅读资产负债表"方法：

| 指标 | 公式 | 含义 |
|------|------|------|
| 生产资产/总资产 | (固定资产+在建工程+无形资产+长期待摊)/总资产 | 重资产 vs 轻资产，重资产退出壁垒高 |
| 应收/总资产 | (应收票据+应收账款)/总资产 | 越高回款越差，>20%需关注 |
| 现金/有息负债 | 现金及现金等价物/有息负债 | 偿债安全边际 |
| 非主业资产/总资产 | (交易性金融资产+投资性房地产+长期股权投资等)/总资产 | 过高说明不专注主业 |

**轻重资产判断**：
- 生产资产/总资产 > 40% → 重资产公司（高固定成本、高退出壁垒、利润波动大）
- 生产资产/总资产 < 40% → 轻资产公司（灵活、护城河往往不来自资产规模）

## 使用原则

1. 优先趋势比较（3-5年）而非单期绝对值
2. 与同行业可比公司对照，区分"行业共性"与"个股异常"
3. 任何单项异常不得直接定性，必须结合利润表和现金流量表交叉验证
4. 关注会计政策变更和合并范围变化对可比性的影响
