# 现金流量表维度检查要点

> 基于《手把手教你读财报》第4章（现金流量表）。用于验证"利润是否转化为现金"、判断企业造血能力与资金链韧性。

## 一、经营活动现金流净额 — 造血能力

- **< 0**：经营现金持续流出，靠外部喂养生存 → 高风险
- **> 0 但长期弱于折旧摊销**：只能维持简单再生产，无力更新设备 → 长期竞争力堪忧
- **明显高于折旧摊销**：有内生的扩张和再投资能力 → 优质

## 二、现金净含量 — 利润含金量

```
现金净含量 = 经营活动现金流净额 / 归母净利润
```

- **连续多年 > 1**：利润较好地转化为现金（排除一次性大额收款后）
- **连续 > 1.2**：非常优质，利润"含金量"高
- **连续 3 年以上 < 1**：高优先级风险信号（R013）。利润可能：
  - 停留在应收账款上（回款困难）
  - 变成了存货（产品积压）
  - 根本不存在（虚增）

需要检查经营现金流净额是否剔除一次性因素（如大额退税、罚款、政府补贴等）。

## 三、销售收现比 — 收入真实性

```
销售收现比 = 销售商品、提供劳务收到的现金 / 营业收入
```

- A股含增值税（13%），理论上此比值应 > 1.13（含增值税销项税）
- 若持续偏离长期历史均值，需排查：
  - 票据结算占比是否大幅增加（银票/商票不计入"销售商品收到现金"）
  - 应收账款是否大幅增加（赊销）
  - 收入确认政策是否改变
- 连续2年以上显著低于合理区间 → R014

## 四、8种现金流量肖像

观察经营/投资/筹资三类活动现金流的符号组合（+为净流入，-为净流出）：

| 序号 | 类型 | 经营 | 投资 | 筹资 | 含义 | 风险 |
|------|------|------|------|------|------|------|
| 1 | 妖精型 | + | + | + | 经营造血、投资回流、仍大举融资 | 警惕：投资回流来自变卖家产？融资目的不明？ |
| 2 | 老母鸡型 | + | + | - | 经营造血、投资回流、偿债/分红 | 优质。需看经营现金流能否覆盖投资+筹资流出 |
| 3 | 蛮牛型 | + | - | + | 经营造血、扩张、融资支持 | 关注投资项目前景和回报周期 |
| 4 | 奶牛型 | + | - | - | 经营造血、扩张、还能还债/分红 | 最优。经营现金流 ≥ 投资+筹资流出之和 |
| 5 | 大出血型 | - | - | - | 全面失血 | 极高风险 |
| 6 | 赌徒型 | - | - | + | 经营失血、还大举投资、靠融资续命 | 高风险 |
| 7 | 混吃等死型 | - | + | - | 经营失血、变卖家产还债 | 高风险 |
| 8 | 骗吃骗喝型 | - | + | + | 经营失血、变卖家产、仍大举融资 | 极高风险 |

**关键判断**：
- 类型 2 和 4 是可持续的良性模式
- 类型 1 和 3 需结合具体情境判断
- 类型 5-8 意味着经营层面存在问题

## 五、投资活动现金流

- **净流出**是正常的（企业在扩张投资），需关注：
  - 投资方向是否与主业相关？（跨界扩张风险大）
  - 投资规模是否超出自身能力？（投资现金流净流出 > 经营现金流净流入 → 依赖融资）
- **净流入**（处置资产回笼 > 新增投资）→ 关注：
  - 是否在变卖家产度日？
  - 处置的是否为核心经营性资产？

## 六、筹资活动现金流

- **大进大出**（流入流出均超过经营现金流净额2倍）→ R027，警惕借新还旧循环
- **持续大额净流入**（经营现金流为负的背景下）→ 依赖外部输血，银行抽贷即崩盘
- **净流出（偿债分红）** + 经营现金流充裕 → 良性

## 七、自由现金流

```
自由现金流 = 经营活动现金流净额 - 购建固定资产、无形资产和其他长期资产支付的现金
```

- 衡量企业可自由支配的现金
- 连续多年为负 → 资本开支过大侵蚀经营现金流，可能"赚了利润没赚钱"
- DCF估值的基础（详见 `checklist-valuation.md`）

## 八、现金流异常信号

- 经营现金流与净利润变动方向相反（净利润增但经营现金流降）
- 经营现金流大增大减（不稳定）
- 投资活动现金流频繁大额流入（在卖什么？）
- 期末现金余额与期初差距巨大但无合理解释

## 使用原则

1. 优先看趋势（3-5期）而非单期判断
2. 必须与资产负债表中的应收、存货、债务联动验证
3. 现金流量表造假难度高于利润表，但并非不可能（可通过现金流分类操纵）
4. 关注附注中现金流量表的补充资料（间接法部分）
