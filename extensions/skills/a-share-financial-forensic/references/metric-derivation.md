# 财报字段到分析指标的加工口径

> 本文件用于“用户提供财报文本时”的字段抽取与指标计算。优先保证可解释性与可复核性。

## 1) 字段抽取优先级

优先提取以下字段（按重要性）：
1. 营业收入
2. 归母净利润
3. 经营活动现金流净额
4. 货币资金
5. 有息负债（短借、长借、应付债券、一年内到期非流动负债等）
6. 应收账款
7. 存货
8. 销售费用、管理费用、研发费用
9. 审计意见、会计政策/估计变更说明

若同一字段存在多个口径：
- 优先使用合并报表口径
- 同时记录口径差异，避免混算

## 2) 指标计算公式

- 现金净含量 = 经营活动现金流净额 / 归母净利润
- 现金债务覆盖 = 现金及现金等价物 / 有息负债
- 应收占收比 = 应收账款 / 营业收入
- 存货成本压力 = 存货增速 - 营业成本增速
- 销售费用率 = 销售费用 / 营业收入
- 管理费用率 = 管理费用 / 营业收入
- 研发费用率 = 研发费用 / 营业收入

## 3) 期间与同比处理

- 明确当期、累计、TTM三种口径，不跨口径直接比较。
- 进行同比比较时，优先使用同口径同期间数据。

## 4) 缺失与异常处理

- 若分母缺失或接近0：不计算该指标，并标注“不可算原因”。
- 若字段疑似单位不一致（元/万元）：先做单位归一再计算。
- 若文本抽取不确定：标注“低置信字段”，不用于强结论。

## 5) 输出中的可追溯要求

每个关键结论需给出：
- 使用的原始字段
- 简化计算过程
- 最终指标结果
- 结论等级（强/中/弱证据）
