# 财报字段到分析指标的加工口径

> 用于从财报文本中抽取字段并计算关键分析指标。优先保证可解释性与可复核性。

## 1) 字段抽取优先级

优先提取以下字段（按重要性）：

| 优先级 | 字段 | 来源 |
|--------|------|------|
| P0 | 营业收入 | 合并利润表 |
| P0 | 归母净利润 | 合并利润表 |
| P0 | 经营活动现金流净额 | 合并现金流量表 |
| P0 | 审计意见类型 | 审计报告 |
| P1 | 货币资金（含受限明细） | 合并资产负债表 + 附注 |
| P1 | 有息负债（短期借款+长期借款+应付债券+一年内到期非流动负债） | 合并资产负债表 + 附注 |
| P1 | 应收账款、应收票据（区分银票/商票） | 合并资产负债表 + 附注 |
| P1 | 存货（含跌价准备） | 合并资产负债表 + 附注 |
| P1 | 商誉 | 合并资产负债表 + 附注 |
| P1 | 固定资产、在建工程 | 合并资产负债表 + 附注 |
| P2 | 销售费用、管理费用、研发费用、财务费用 | 合并利润表 + 附注 |
| P2 | 资产减值损失 | 合并利润表 + 附注 |
| P2 | 非经常性损益 | 年报"非经常性损益"专项披露 |
| P2 | 大股东股权质押、冻结信息 | 年报/季报公告 |
| P2 | 会计政策与估计变更说明 | 附注 |
| P3 | 毛利率（计算所得） | 利润表 |
| P3 | 所得税费用、递延所得税 | 附注 |

若同一字段存在多个口径：
- 优先使用合并报表口径
- 记录口径差异，避免混算

## 2) 指标计算公式

### 偿债能力
- **现金债务比** = 现金及现金等价物 / 有息负债合计
  - 有息负债 = 短期借款 + 长期借款 + 应付债券 + 一年内到期非流动负债 + 长期应付款（融资租赁部分）
- **有息负债率** = 有息负债 / 总资产

### 营运能力
- **应收周转天数** = 应收账款平均余额 / 营业收入 × 365
- **存货周转天数** = 存货平均余额 / 营业成本 × 365
- **总资产周转率** = 营业收入 / 总资产平均余额

### 盈利质量
- **现金净含量** = 经营活动现金流净额 / 归母净利润
- **销售收现比** = 销售商品提供劳务收到的现金 / 营业收入
  - 注：A股增值税率13%，含税口径理论值 > 1.13
- **毛利率** = (营业收入 - 营业成本) / 营业收入
- **营业利润率** = 营业利润 / 营业收入
- **净利率** = 归母净利润 / 营业收入

### 费用率
- **销售费用率** = 销售费用 / 营业收入
- **管理费用率** = 管理费用 / 营业收入
- **研发费用率** = 研发费用 / 营业收入
- **财务费用率** = 财务费用 / 营业收入

### 风险指标
- **应收/营收** = 应收账款 / 营业收入
- **商誉/净资产** = 商誉 / 归母净资产
- **非经常损益占比** = 非经常性损益 / 归母净利润
- **存货与成本增速差** = 存货增速 - 营业成本增速

### 杜邦分析
- **ROE** = 归母净利润 / 归母净资产
- **ROE** = 净利润率 × 总资产周转率 × 杠杆系数
  - 净利润率 = 归母净利润 / 营业收入
  - 总资产周转率 = 营业收入 / 总资产
  - 杠杆系数 = 总资产 / 归母净资产

### 资产结构
- **生产资产占比** = (固定资产 + 在建工程 + 无形资产 + 长期待摊费用) / 总资产
- **非主业资产占比** = (交易性金融资产 + 投资性房地产 + 长期股权投资 + 可供出售金融资产等) / 总资产

## 3) 期间与同比处理

- 明确当期、累计、TTM三种口径，不跨口径直接比较
- 同比比较：优先使用同口径同期间数据（如2025Q1 vs 2024Q1）
- TTM（Trailing Twelve Months）：最近4个季度滚动加总，平滑季节性影响
- 注意：新会计准则切换可能造成前期数据不可比，需单独标注

## 4) 缺失与异常处理

- 分母缺失或接近0 → 不计算该指标，标注"不可算原因"
- 字段疑似单位不一致（元/万元/亿元）→ 先做单位归一再计算
- 文本抽取不确定 → 标注"低置信字段"，不用于强结论
- 同一指标不同来源冲突 → 标注差异，优先采信审计后数据

## 5) 输出可追溯要求

每个关键结论需给出：
- 使用的原始字段（来源行/科目）
- 简化计算过程
- 最终指标结果
- 结论等级（强证据/中证据/弱证据）

示例：
```
疑点：R013 经营现金流净额/净利润持续<1
- 2023年：经营现金流净额 5.2亿 / 归母净利润 8.1亿 = 0.64
- 2024年：经营现金流净额 3.8亿 / 归母净利润 9.3亿 = 0.41
- 2025年：经营现金流净额 2.1亿 / 归母净利润 10.5亿 = 0.20
- 连续3年<1，且趋势恶化
- 结论等级：强证据
```
