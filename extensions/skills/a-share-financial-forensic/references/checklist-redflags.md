# 红旗与治理风险检查要点

> 基于《手把手教你读财报》第8章（欺诈与反欺诈）及本书中散落各章节的避雷信号和重要提示整理。按严重度分高/中/低三级。

## 一、审计与治理层面（最高优先级，发现即标记）

### 审计意见
- **非标准无保留意见**（保留意见/无法表示意见/否定意见）→ R019，直接高风险
  - 会计师用"除...外"表述 → 保留意见
  - 会计师说"无法获取充分适当的审计证据" → 无法表示意见
  - 会计师说"财务报表存在重大错报" → 否定意见
- 带强调事项段的无保留意见（持续经营能力存在重大不确定性等）→ 虽然不是保留意见，但需高度关注

### 审计机构
- 3年内更换会计师事务所超过1次（非强制轮换）→ R020，可能是：
  - 会计师发现重大问题不愿签字
  - 公司"购买"更配合的审计意见
- 由非知名小所审计但公司业务复杂/海外收入占比高 → R031

### 高管变动
- 财务总监、董秘等关键岗位在年报前后异常离职 → 可能是"逃逸"信号
- 独董/监事公开提出异议或发表保留意见

### 信息披露
- 未按时披露定期报告
- 存在被证监会/交易所立案调查、处罚记录
- 会计差错更正频繁

## 二、会计政策与估计变更

- **固定资产折旧政策显著变更**（如突然延长折旧年限）→ 直接美化利润
- **坏账计提标准异常宽松**或当年突然降低计提比例 → R004、R037
- **通过减值/转回在年度间平滑利润** → R038

## 三、资金与资金占用

- **存贷双高**（货币资金 + 有息负债同时畸高）→ R010
- **受限资金占比 > 30%**但披露不充分 → R012
- **预付款/保证金/其他应收款异常增长** → 大股东资金占用的常见通道：
  - 预付款→关联方→无真实业务→资金被占用
  - 其他应收款→大股东/关联方→长期挂账不还

## 四、收入与利润真实性风险

- 毛利率异常波动（>10个百分点）且无合理解释 → R032
- 毛利率明显高于同行（>20个百分点）且无护城河支撑 → R015
- Q4营收占比异常偏高且无季节性原因 → R023
- 海外业务占比高 + 利润率异常 + 审计所非知名 → R029、R031
- 关联交易占比畸高 → R022
- 虚构合同+资金循环（资金转出→体外循环→回流伪造收入）→ R033
- 在建工程资金体外循环（承包方为关联方，款项无真实工程对应）→ R034

## 五、长期资产与并购风险

- 商誉占比高 + 减值测试不充分 → R008、R009
- 在建工程长期不转固 → R024
- 资产减值转回金额大（当期转回/净利润 > 10%）→ R038

## 六、大股东行为

- **大股东股权质押比例 > 50%** → R021，警惕：
  - 大股东资金紧张，可能挪用上市公司资金
  - 股价下跌引发平仓风险，控制权不稳
- **大股东股权被司法冻结** → R030，直接高风险
- 大股东频繁减持套现

## 七、经营与现金流红旗

- 连续3年经营现金流净额为负 → R035
- 经营现金流净额/净利润连续3年以上 < 1 → R013
- 销售收现比持续显著偏低 → R014

## 严重度分级建议

### 高风险（一票否决级）
- 审计意见非标
- 大股东股权被冻结
- 经营现金流连续3年为负 + 存贷双高
- 会计师事务所频繁更换（3年>1次）
- 在建工程长期不转固 + 超预算100%
- 虚构交易/资金循环明确证据

### 中风险（需进一步验证）
- 单项财务比率异常（如应收/营收>30%）
- 大股东质押>50%
- 毛利率异常但缺乏明确造假证据
- 非经常性损益占比持续偏高
- Q4营收季节性异常

### 低风险（持续跟踪）
- 轻微偏离行业均值
- 单一指标小幅恶化
- 会计政策正常范围内调整

## 八、外部信号（书中第8章提及）

除了报表内部信号，还需关注：
- 媒体质疑或做空报告
- 监管部门问询函
- 同行业可比公司数据是否与行业趋势一致（全行业萧条但公司一枝独秀 → 警惕）
- 前员工/供应商/客户举报
- 税务数据与财务数据不一致（"税报"vs"财报"）

## 使用原则

1. 硬伤信号（审计、资金占用、治理异常）一旦确认，优先于其他分析
2. 多项中风险信号叠加 → 升级为高风险
3. 红灯信号不自动等于造假，但需要公司给出具体、可验证的解释
4. 证据不足时标注"存疑"而非强行定性
