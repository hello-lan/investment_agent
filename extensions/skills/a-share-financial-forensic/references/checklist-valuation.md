# 估值维度检查要点

> 基于《手把手教你读财报》第5章中的估值方法和三大前提。用于形成"前提显式"的估值观察，不输出交易指令。

## 一、估值三大前提

在进行任何估值前，必须先确认三个前提：

1. **利润是否为真？**
   - 经营现金流是否持续验证净利润？
   - 如利润存疑，估值无意义

2. **利润是否可持续？**
   - 利润来源是主业还是非经常项目？
   - 公司是否具有护城河（品牌、技术、网络效应、规模优势等）？

3. **维持当前盈利是否需要大量资本投入？**
   - 每年需要多大比例的利润再投入才能维持现有盈利能力？
   - 资本开支是否持续侵蚀自由现金流？

**若任一前提不满足或存疑，估值只能给区间参考，置信度需下调。**

## 二、估值方法

### 1. 自由现金流折现法（DCF）

适用于非周期类、现金流稳定的公司。

核心思路：企业价值 = 未来自由现金流的折现值之和。

- 自由现金流 = 经营现金流净额 - 维持性资本支出
- 折现率参考：无风险利率（10年期国债）+ 风险溢价
- 关键假设：未来增速、折现率、永续增长率，任何一个微调都会显著影响结果

> DCF对假设极其敏感，实际使用时给出"乐观/中性/悲观"三个情景而非单一数值。

### 2. 清算价值法

适用于重资产公司或困境公司。

- 以资产负债表为基础，对各资产科目打折估算可变现价值
- 货币资金 = 100%，应收 = 70-80%，存货 = 50-70%，固定资产 = 30-50%
- 减去全部负债后的余值为清算价值

### 3. 市场比较法

- PE（市盈率）：最常用，但对周期股在周期高点给低PE反而危险
- PB（市净率）：适用于金融、资源类公司
- PS（市销率）：适用于高增长但未盈利公司

比较时注意：
- 不同行业的合理估值区间差异巨大
- 同一行业不同公司的估值差异可能反映护城河差异
- 牛市和熊市的整体估值中枢不同

## 三、周期类公司特殊处理

周期类公司（如钢铁、化工、航运、生猪等）：
- 绝不能以景气高点利润 × 合理PE来估值（陷阱：高利润时PE最低 → "估值便宜"的假象）
- 应用10年平均利润平滑周期波动
- 在周期低谷（公司亏损、PE为负）时可能是最好的投资时机

## 四、估值输出格式

估值分析用以下结构输出：

```
### 估值前提判断
- 利润为真：是/否/存疑
- 利润可持续：是/否/存疑
- 低资本投入维持：是/否/存疑

### 估值方法选择
- 适用方法：DCF / 清算 / 比较
- 关键假设：增速X%、折现率Y%

### 估值区间（仅供参考，不构成投资建议）
- 乐观情景：X元/股
- 中性情景：Y元/股
- 悲观情景：Z元/股

### 估值信息不足项
- 缺失字段1（影响：...）
- 缺失字段2（影响：...）
```

## 五、估值注意事项

1. 估值必须携带前提条件说明
2. 给出区间而非精确值，承认不确定性
3. 关键变量缺失时输出"估值信息不足 + 需补字段"
4. 绝对不输出"买入""卖出""目标价"等交易指令
5. 估值结果应表述为"在X前提下，XX元的估值区间属于合理/偏高/偏低"
6. DCF对永续增长率和折现率的微小变化极度敏感，不要产生"精确"的错觉

## 使用建议

- 估值是财务分析的最后一环，先完成"排雷"再做估值
- 对存疑公司（利润真实性存疑）不做DCF估值
- 跨行业比较时优先用PB或PS而非PE
